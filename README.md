# klasifikasi-teks-mbg
Sistem Klasifikasi Teks Opini Program Makan Bergizi Gratis (MBG) dari X/Twitter menggunakan Pembobotan TF-IDF dan Multi-Classifier Machine Learning.
### 📊 Dataset Proyek

Seluruh data opini publik hasil *scraping* dari platform X (Twitter) disimpan secara aman di Google Drive dan terbagi menjadi 3 berkas dataset berkala. Anda dapat mengunduh masing-masing dataset melalui tautan resmi di bawah ini:

*   [📥 Dataset 1 (Makan Bergizi Gratis - Batch A)](https://drive.google.com/file/d/1KAAIuNk1P37fHTYPoaCjQHrIk1UkOQlG/view?usp=sharing)
*   [📥 Dataset 2 (Makan Bergizi Gratis - Batch B)](https://drive.google.com/file/d/13jkE8nXKtlOW6SGsYyzJMpVaxfyCFFCv/view?usp=sharing)
*   [📥 Dataset 3 (Makan Bergizi Gratis - Batch C)](https://drive.google.com/file/d/1n1QBv7AkmyPaALS5gn5k0tF64m4t26TR/view?usp=sharing)

> **Catatan**: Pastikan Anda memasukkan ketiga file ini ke dalam folder penyimpanan Google Drive Anda apabila ingin menjalankan *pipeline* penggabungan data (`pd.concat`) secara mulus langsung pada Jupyter Notebook (`.ipynb`).
